About This App

This is a simple Flask web application that demonstrates basic web development concepts including:

  • Routing: Different URLs that serve different pages
  • Templates: HTML templates with Jinja2 templating
  • Forms: Handling user input through web forms
  • Flash Messages: Displaying feedback to users
  • Basic CRUD: Create, Read, Update, Delete operations
  • API Endpoint: JSON API for accessing data

Features

This task manager app includes:

  • Add new tasks
  • Mark tasks as completed
  • Delete tasks
  • View task statistics
  • Responsive design

Technical Details

  • Framework: Flask (Python)
  • Templating: Jinja2
  • Styling: Custom CSS
  • Data Storage: In-memory (list)

Note: This is a demonstration app. In production, you would use a proper database and implement additional security measures.

API Access

You can also access the tasks data via JSON API:

GET /api/tasks
